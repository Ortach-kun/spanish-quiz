Background Music Instructions

1. Place your MP3 file in this directory (/public/audio/).
2. The current filename is "backround-music.mp3". Make sure your MP3 file uses this name or update the source path in the HTML file.
3. The audio will play automatically when the "Start Quiz" button is clicked.
4. The music will continue playing throughout the quiz with no option to turn it off.

Note: Due to browser policies, audio can only be played after user interaction with the page, such as clicking the "Start Quiz" button.

If you have issues with audio playback, try:
- Using a different browser
- Checking that your MP3 file is valid and not corrupted
- Ensuring your device volume is turned up
